# SmartTools — Japanese Stock Analyst

A local MCP tool, exposed publicly via a tunnel, that Claude calls to produce bull/base/bear price scenarios for Tokyo Stock Exchange stocks. Driven from a small Tkinter desktop app.

## Layout

- `src/mcp_server/` — FastMCP server hosting the `analyze_japanese_stock` tool (Claude API + yfinance).
- `src/gui_client/` — Tkinter app that calls the Anthropic Messages API with `mcp_servers` pointing at the public tunnel.
- `tests/` — pytest unit tests.

## One-time setup

```bash
# from C:\Projects\SmartTools
.venv\Scripts\python.exe -m pip install -e ".[dev]"
```

Edit `.env` and confirm:
- `ANTHROPIC_API_KEY` — from https://console.anthropic.com
- `MCP_BEARER_TOKEN` — long random string (already generated)
- `MCP_PORT` — default 8765
- `PUBLIC_MCP_URL` — filled in once your tunnel is up (see below)

## Run the MCP server (terminal 1)

```bash
.venv\Scripts\python.exe -m mcp_server.server
```

This binds to `http://127.0.0.1:8765/mcp` and requires `Authorization: Bearer $MCP_BEARER_TOKEN`.

## Expose it publicly (terminal 2)

Install [cloudflared](https://github.com/cloudflare/cloudflared/releases) once:

```bash
winget install --id Cloudflare.cloudflared
```

Start a quick tunnel:

```bash
cloudflared tunnel --url http://127.0.0.1:8765
```

Copy the printed `https://<random>.trycloudflare.com` URL, append `/mcp`, and put it in `.env` as `PUBLIC_MCP_URL`. The tunnel URL changes each run — for a stable URL, use a [named tunnel](https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/get-started/create-remote-tunnel/) tied to your Cloudflare account.

## Run the GUI (terminal 3)

```bash
.venv\Scripts\python.exe -m gui_client.app
```

Enter a TSE code (e.g. `7203` Toyota, `9984` SoftBank, `6758` Sony) and a horizon in days; click **Analyze**.

## Run tests

```bash
.venv\Scripts\python.exe -m pytest -q
```

## Disclaimer

The tool's output is generated by an LLM and is **not investment advice**.
